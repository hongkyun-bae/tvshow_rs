## Notice
Only selected files from MyMediaLite are included.
The project is not executable with these files alone; they are provided solely to illustrate the core implementation logic.

- Program.cs: Main entry point of the program.
- Scenario.cs: Returns the model object based on the specified configuration.
- BPRMF.cs: Implements BPR-MF training and generates predicted preference scores for each TV program per user.
